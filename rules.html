Sudoku

  1. Put one number from 1 to N to each square. N is size of puzzle.
  2. All numbers in each row and column must be different.
  3. All numbers in each rectangle must be different.

Sudoku without rectangles

  1. All Sudoku rules except Sudoku 3 apply.

Diagonal sudoku

  1. All Sudoku rules apply.
  2. All numbers in each of two diagonals must be different.

VX sudoku

  1. All Sudoku rules apply.
  2. All pairs of orthogonally adjacent squares with sum equal to one of specific values are labeled with character specific to that value.
  3. Default specific sum values are 5 and 10, their specific characters are "V" and "X".

Kropki sudoku

  1. All Sudoku rules apply.
  2. All orthogonally adjacent squares, where the difference of numbers is 1, have white square between them.
  3. All orthogonally adjacent squares, where the number in one is two times greater than the other, have black square between them.
  4. 1/2 can have white or black square between them.
  • Does not include determining unambiguity of solution, therefore any task can have multiple solutions.
  • Color of 1/2 is determined in method Renderer.render(). Multiple calls of this function will sometimes create different tasks.

Easy as ABC

  1. Write one or none of first N letters of alphabet in each square. N is number written in top-left square of puzzle.
  2. All letters in each row or column must be different. There can be multiple squares without any number.
  3. Numbers on the 4 sides of the puzzle are first letters from that direction.
