How to use

  1. Get a javascript console on your internet explorer.
  2. Go to page "Code" and copy some javascript code there.
  3. Go to page "Generator" and put the code in your javascript console.
  4. Wait for the puzzles to generate.
  5. Press ctrl + P (as you do for printing), but save the puzzles into a .pdf file.

Github

Repository with typescript code https://github.com/simonbrecher/sudoku branch main.

You need to compile typescript code into javascript code, if you want to use the source code from github. https://www.jetbrains.com/help/webstorm/compiling-typescript-to-javascript.html

To generate mastermind, you need to add file words.txt with a lot of words (one word per line, lowercase).
