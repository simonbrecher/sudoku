Sudoku

  1. Put one number from 1 to N to each square. N is size of puzzle.
  2. All numbers in each row and column must be different.
  3. All numbers in each rectangle must be different.

Sudoku without rectangles

  1. All Sudoku rules except Sudoku 3 apply.

Diagonal sudoku

  1. All Sudoku rules apply.
  2. All numbers in each of two diagonals must be different.

Sudoku with irregular groups

  1. All Sudoku without rectangles rules apply.
  2. All numbers in each irregular groups must be different.

VX sudoku

  1. All Sudoku rules apply.
  2. All pairs of orthogonally adjacent squares with sum equal to one of specific values are labeled with character specific to that value.
  3. Default specific sum values are 5 and 10, their specific characters are "V" and "X".

Kropki sudoku

  1. All Sudoku rules apply.
  2. All orthogonally adjacent squares, where the difference of numbers is 1, have white square between them.
  3. All orthogonally adjacent squares, where the number in one is two times greater than the other, have black square between them.
  4. 1/2 can have white or black square between them.
  • This variation has "K" on top-left.
  • Does not include determining unambiguity of solution, therefore any task can have multiple solutions.
  • Color of 1/2 is determined in method Renderer.render(). Multiple calls of this function will always create same task.

Minus one sudoku

  1. All Sudoku rules apply.
  2. All orthogonally adjacent squares, where the difference of numbers is 1, have white square between them.
  • This variation has "-" on top-left.

Inequality sudoku

  1. All Sudoku rules apply.
  2. All inequalities of orthogonally adjacent squares are shown.

Killer sudoku

  1. All Sudoku rules apply.
  2. Numbers in dashed groups of squares must have sum same, as the number written on top left of that group.

Easy as ABC

  1. Write one or none of first N letters of alphabet in each square. N is number written in top-left square of puzzle.
  2. All letters in each row or column must be different. There can be multiple squares without any number.
  3. Numbers on the 4 sides of the puzzle are first letters from that direction.

Skyscrapers

  1. All Sudoku without rectangles rules apply.
  2. Letters represent heights of skyscrapers. A skyscraper is visible, if no higher skyscraper is before him.
  3. Letters on the 4 sides of the puzzle are number of skyscrapers visible from that direction.

King move

  1. All pairs of squares, which are king move from each other, must not contain same numbers, or letters in them.
  • This variation has "K" in top-right square.

Knight move

  1. All pairs of squares, which are knight move from each other, must not contain same numbers, or letters in them.
  • This variation has "N" in top-right square.

Coral

  1. Color some squares black.
  2. There can not be 2x2 block of black squares.
  3. All black squares must be orthogonally connected.
  4. All white squares must be orthogonally connected to the side.
  5. Numbers on the 2 sides of the puzzle are lengths of all connected blocks of black squares in correct order.

Sorted coral

  1. All Coral rules except Coral 5 apply.
  2. Numbers on the 2 sides of the puzzle are lengths of all connected blocks of black squares, not necessarily in correct order.
  • This variation has "S" in the top-left square.

Dominos

  1. Divide board into 1x2 domino pieces.
  2. Each piece must be used exactly once.

Galaxies

  1. Divide board into groups.
  2. Each square must be in exactly one group.
  3. All squares in each group must be orthogonally connected.
  4. Each group must be centrally symmetric, where the center of symmetry is one of the small white squares.
  5. Each small white square must belong to exactly one group.
  6. Each group must contain all squares, that its center of symmetry overlaps with.

Mastermind

  1. Write one letter in each empty square in last line. Same letters can repeat.
  2. Only letters that are at least once in task can be in solution.
  3. Black and white dots next to words in task are created by comparing each word to task.
  4. Black dot is given or each position in word, that has same letter as solution.
  5. For all types of letters is counted its frequency in word and solution. (only on positions that were not given black dot already) For each type of letter will be given as many white dots as is smaller number from its frequency in word and its frequency in solution.

Kuromasu

  1. Color some squares black.
  2. Two black squares can not be orthogonally adjacent.
  3. All white squares must be orthogonally connected.
  4. From each square with number must be visible that many white squares in all 4 directions. (including the square with the number)
  • This variation has "K" in the top-left square.

Cave

  1. Color some squares black.
  2. All black squares must be orthogonally connected to the side.
  3. All white squares must be orthogonally connected.
  4. From each square with number must be visible that many white squares in all 4 directions. (including the square with the number)
  • This variation has "C" in the top-left square.

Slitherlink

  1. Draw lines between some vertically or horizontally neighbouring small squares.
  2. All lines must create a single loop.
  3. The loop must not intersect or touch itself.
  4. Numbers say, how many lines there are on 4 closes sides around that number.

Stars

  1. Draw one star in some squares.
  2. Stars must not be on orthogonally or diagonally adjacent.
  3. There must be exactly N stars in each row, line and region. N is ⌊(M + 1) / 5⌋, where M is size of the puzzle.

Tapa

  1. Color some squares black.
  2. Squares with numbers must not be black.
  3. There must not be 2x2 block of black squares.
  4. All black squares must be orthogonally connected.
  5. All white squares must be orthogonally or diagonally connected to side.
  6. Digits of numbers in squares represent lengths of connected blocks of black squares in 8 squares next to the square with the number.
